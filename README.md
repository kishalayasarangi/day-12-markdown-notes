# Day 12 — Markdown Notes App

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A fully featured markdown notes app that runs in the browser.
Notes are saved in localStorage so they persist between sessions.

## How to view
Open `index.html` in your browser.

## Features
- Create, edit, delete notes
- Markdown rendering with live preview
- Toolbar for bold, italic, heading, list, code, link
- Search notes
- Word and character count
- Auto-save to localStorage
